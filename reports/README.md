# Test Reports

This directory contains test execution reports generated by Maven Surefire and Karate.

## Report Files

- **surefire-reports/**: Contains XML and HTML test reports from Maven Surefire
  - `TEST-*.xml`: Individual test class results
  - `testng-results.xml`: TestNG suite results
  - `index.html`: HTML report (if generated)

## Generating Reports

To generate fresh test reports, run:

```bash
./generate-reports.sh
```

Or manually:

```bash
mvn clean test surefire-report:report
```

## Viewing Reports

Open the HTML reports in your browser:
- Surefire HTML Report: `reports/surefire-reports/index.html` (if available)
- Or view the XML files directly

## Note

These reports are tracked in git to share test results with the team. 
Regenerate them after running tests to keep them up-to-date.
